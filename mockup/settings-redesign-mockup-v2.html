Tab 1 — General: Unchanged from v1. Profile, display, capture region, calibration. Setup-once tab.
Tab 2 — Detection: Now has three main sections. Cooldown Thresholds — darken, trigger, change slider (new), change ignore field (new). Glow / Ready Override — the big new block: enable checkbox at top, then three subsections (Glow Parameters as a compact 4-col grid, Per-Slot Overrides for the text fields, Color Thresholds for the sliders and hue ranges). When the enable checkbox is off, all subsections dim. Cast / Channel Detection — same as v1 with subsections for Cast Band and Cast Timing, dimmed when disabled. Cast Bar Region — separate section for the ROI override.
Tab 3 — Automation: Unchanged from v1. Controls, timing, priority lists, spell queue.
Status bar: Replaces Save Config button. Shows auto-save status with a green dot + relative time ("Last saved: 2m ago", "Last saved: just now"). Dot turns amber during active save. Sits outside the tab widget so it's always visible. Updates every ~30s or on save event.
Glow layout strategy: The glow settings are numerous but logically group into three concerns: (1) the ring/V+/S/N parameters that control how glow is detected, (2) per-slot text fields for exceptions, and (3) color fraction sliders + hue ranges. Each gets its own subsection with a darker background, keeping the section scannable. The 4-column grid for Ring/V+/S≥/N and Y min/Y max/R≤/R≥ is compact because these are small numeric values that belong together.
